DON’T SKIP LEG DAY#

This assignment is like a workout. You’re going to focus on specific topic areas and work at your comfort level, building up until you’ve reached your personal limit.

Each section of the assignment adds and focuses on a different topic and within each section the questions build from introductory to master.

  • Focus on one section at a time, working in order. In each section, start where you feel comfortable and work up until you max out.

  • You must complete at least two problems from each section (and ten problems total from the assignment).

OR

  • complete the challenge problem for the section. Challenge problems are not limited to just the tools of that section.

Functions and Variables#

Problem 1.1

Write the function middle that takes three arguments a, b, and c and returns the value of b.

For example middle(1,2,3) returns 2. Also, middle(a = 1, c=2, b=3) returns 3.

Problem 1.2

Write the function say_hi that takes as input a string and prints a message that says “Hi” to whatever is the string. The function should not return any value.

For example, say_hi('Ralphie') prints the message “Hi Ralphie”.

Problem 1.3

Write the function what_is that takes as input anything and returns its type. The built in function type() returns the type of any Python object. So basically, you are wrapping the function type() in another function giving it a new name.

Problem 1.4

Write the function add that takes two numbers and returns their sum. If either number is missing, the default value should be 0.

What happens if you give your function two strings as inputs? What if you only give one string as input?

Challenge Problem 1.5

Write a function missing_inputs that can take up to five inputs either strings or numbers and returns two outputs, the sum of the numbers and the sum of the strings. If any input is missing, prompt the user to provide the missing input.

Conditional Statements#

Problem 2.1

Write the function letter_grade that takes as input a number between 0-100 and returns the letter grade.

Problem 2.2

Write the function calculator that takes three inputs operation (a string), a (a number), and b(another number).

The argument operation should expect one of the following four strings add, subtract, multiply, or divide and return either a+b, a-b, a*b, or a/b respectively. If operation is a different string, print a message and return nothing (just return).

Problem 2.4

Write the function convert_units that takes as input a number and two strings corresponding to units of measurement in imperial and metric systems. Your function should convert the measurements from unit_1 to unit_2. Your function should be able to do the conversions in both directions.

The unit pairs should be:

  • 1 lb = 0.454 kg

  • 1 in = 2.54 cm

  • temp F = (temp - 32)/1.8 C

Problem 2.4

Write the function is_that_a_question that takes as input a string and returns True or False based on whether it thinks the string is a question.

Challenge Problem 2.5

Write the function which_triangle that takes three inputs corresponding to the lengths of the sides of the triangle. Print a string containing the adjectives ‘equilateral’, ‘isosceles’, ‘scalene’, ‘right’, or ‘Not possible’. The printed output may have more than one of these words.

  • equilateral - all sides equal

  • isosceles - two sides equal

  • scalene - no sides equal

  • right - one angle is 90 (test Pythagorean theorem)

  • not possible - the sum of the shorter two sides is less than the longest side

Lists#

Problem 3.1

Write the function first_n that takes as input a list and a number n and returns the first n items of the list.

Problem 3.2

Write a function pad_zeros that takes as input a list and a number. The function should add the specified number of zeros to the beginning and end of the list. If no number is given, pad the list with one zero on each end.

For example pad_zeros([1,3,9], 2) should return the list [0, 0, 1, 3, 9, 0, 0].

Problem 3.3

Write the the function skip_n that takes as input a list and a number n and returns a list with every nth element starting from the beginning of the list.

Problem 3.4

A palindrome is a list or word that reads the same forward and backwards.

Write the function make_palindrome that takes a list and attaches the reverse of the list to the end making a palindrome, but does not double the center letter!

Challenge Problem 3.5

Write the function make_shortest_palindrome that takes a list and adds elements to either the beginning or end or both to make the shortest palindrome list possible. If the list is already a palindrome, your job is done.

Loops#

Problem 4.1

Write the function print_list that takes as input a list and prints a numbered list with each element on its own line. If you want to be fancy, look up how to use enumerate.

Problem 4.2

Write the function run_sum that takes as input a list of numbers and returns a list containing the running (cumulative) sum of the numbers.

For example run_sum([1, 2, 3, 4]) should return the list [1, 3, 6, 10].

What happens if you give your function a list of strings?

Problem 4.3

Write the function item_lengths that takes as input a list of things. The things in the list will each be either a number, a string, or a list. The function should return a list with the corresponding length of the item. If the item is a number, its length should be 1.

For example item_lengths([4, 'hello', [1, 3], ['a', 'b', 'c', 'dog']]) should return the list [1, 5, 2, 4].

Problem 4.4

Write the function hi_lo that has the player guess a secret number between 0-100. Ask the user for to input a number between 1-100. Check that the input is a number and is in that range. If not have them guess again. When the user inputs a number, reply whether the secret number is ‘much higher’ (more than 10), ‘higher’, ‘lower’, ‘much lower’ (more than 10) or ‘exactly right’. End the loop when the player guesses the correct number.

Problem 4.5

A Caesar cipher is a simple encryption that shifts every letter by some fixed number. For example, ABC shifted by 2 would be CDE.

Write the function caesar that takes as input a string and a number and encodes the string using the shift of that mumber. Keep all capitalization and punctuation the same.

Test your function by encrypting a string. You can decipher your encrypted string by using the function with the negative of the shift number. For example, if you use caesar('hello', 4) to encrypt to get “lipps”, caesar('lipps', -4) should return “hello”.

Challenge Problem 4.6

Write the function factorize that takes as input an num (an integer) and returns all the prime factor decomposition (including duplicates) as a list.

For example factorize(12) should return [2,2,3].

Dictionaries#

Problem 5.1

Write the program split_dict that takes as input a dictionary and returns two lists, the first containing the keys of the dictionary and the second containing the values.

Problem 5.2

Write the program make_dict that takes as input two lists, the first list contains only strings and will be the keys to the dictionary and the second list contains anything and will be the values. If the lists are not the same length, either: keep going until you run out of keys or give extra keys the value None if you run out of values.

Problem 5.3

Write the function print_dict that takes as input a dictionary and prints each key-value pair on a new line.

Problem 5.4

Write the function thing_counter that takes as input a block of text and returns a dictionary. Whenever a number appears in the text, if it is followed by a word, that word should be added to the dictionary as a key and the value should be the number. If the word is already in the dictionary, increment the count.

Useful tool: .split() acting on a string will segment the string wherever there is a space and return a list of substrings. To break a string on a different separating character(s), use .split(separator).

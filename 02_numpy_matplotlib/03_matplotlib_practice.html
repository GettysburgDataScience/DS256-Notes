11.1. Visualizing NFL QB contracts#

Many of the datasets used in class notebooks are stored in the Gettysburg Datascience datasets github repository.

From this page, you can select a dataset from the list to see a preview of the data (provided that the dataset is not too large).

To get the data, we’ll need to import it in its raw form as a .csv.

  • Click the RAW button in the upper right of the data menu bar.

  • You should then see the data represented as plain text.

  • Copy the URL of that page. That is the location of the data.

11.1.1. Downloading the data#

After finding the data URL, we use np.genfromtxt() to bring the data into numpy.

Remember, numpy arrays may only contain one data type! So first, we’ll look at the headers to get a sense of the data each column contains. Column headers are generally in the top row of the data file.

Here is the np.genfromtxt documentation

qb_contracts_git = 'https://raw.githubusercontent.com/GettysburgDataScience/datasets/refs/heads/main/qb_contracts.csv'
qb_contracts_headers = np.genfromtxt(qb_contracts_git, delimiter = ',', dtype=str, max_rows=1)

                     dtype=np.datetime64,
                     delimiter = ',')

tmax = np.genfromtxt(gettysburg_weather_git, 
                     usecols = 4, skip_header=1,
                     dtype=int, missing_values = np.nan,
